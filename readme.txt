To run the code with default values, input the following into a terminal:

python3 sender.py

=============================================

To run the code with custom values, specify appropriate values for the -f, -a, -s, -c, and -i flags. For example:

python3 sender.py -f 2b97c5ee.txt -a 10.0.7.141 -s 9000 -c 6750 -i 2b97c5ee

where

-f - pathname of the file to transmit
-a - IP address of the receiver
-s - port number that the receiver will use
-c - port number that the sender will use
-i - unique ID assigned to the student

=============================================

You may also use "python" instead of "python3" depending on your environment.
